# Simple Interest Calculator

This repository contains a simple Bash script that calculates simple interest.

## Formula

Simple Interest = (Principal × Rate × Time) / 100

Where:

* **Principal (P):** The initial amount of money.
* **Rate (R):** Annual interest rate (in percent).
* **Time (T):** Time period (in years).

## Files

* `simple-interest.sh` – Bash script to calculate simple interest.
* `LICENSE` – Open source license.
* `README.md` – Project documentation.

## How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/<your-github-username>/<repository-name>.git
   ```

2. Change to the project directory:

   ```bash
   cd <repository-name>
   ```

3. Make the script executable:

   ```bash
   chmod +x simple-interest.sh
   ```

4. Run the script:

   ```bash
   ./simple-interest.sh
   ```

## Example

**Input**

* Principal = 1000
* Rate = 5
* Time = 2

**Output**

* Simple Interest = 100
